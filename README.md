# Advisor Panel

Endpoints: Below endpoints we are using to communicate with node server

1. login
   - POST - auth/login
2. signup
   - POST - auth/register
   - POST - auth/checkEmail
3. forgotPassword
   - POST - auth/forgotPass
   - POST - auth/checkEmail
   - POST - auth/matchCode
4. profile
   - GET - user/info
   - user/info
5. resetPassword
   - PUT - auth/resetPassword
